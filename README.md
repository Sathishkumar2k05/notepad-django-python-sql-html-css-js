Title: Notepad Full-Stack Web Application
Tech Stack: Python, Django, MySQL Workbench, HTML, CSS, JavaScript (Dark Mode), Class-Based Views (CBV)

Description:
Developed a full-stack Notepad web application that allows users to create, edit, view, and delete notes. Implemented the back-end using Django with Class-Based Views to manage CRUD operations efficiently. Integrated MySQL Workbench as the database for storing user notes securely. Designed the front-end using HTML and CSS, and added JavaScript for interactive features like Dark Mode toggle.

Key Features:

Create, update, delete, and view notes (CRUD)

User-friendly interface built with HTML & CSS

Dark and Light theme switch using JavaScript

Class-Based Views for clean and structured backend code

MySQL database integration using MySQL Workbench

Responsive UI for better user experience


Responsibilities:

Designed and developed both front-end and back-end.

Created database tables and handled SQL queries.

Implemented CBV for modular and maintainable code.

Managed routing, templates, and static files in Django.

Tested each feature and optimized performance.
